## NXP GREENBOX II VEHICLE ELECTRIFICATION DEVELOPMENT PLATFORM

- Development platform for NXP S32S247TV, an Arm® Cortex® R52 safe and secure SoC
- High-performance ASIL D compute with 4 x Arm® Cortex®-R52 cores with Neon SIMD running in lockstep and operating at up to 800 MHz
- Peripheral control using complex timers, filters and analog modules to support Hybrid Electric Vehicle (HEV) or motor control applications
- Numerous automotive communication interfaces: Ethernet, CAN FD, LIN, UART, JTAG, SDHC, PSI5, SENT
- Hypervisor support for virtual machines and task isolation
- Easy to use out of the box experience with S32 Design Studio IDE and sample software for advanced control applications

## S32G GOLDBOX SERVICE-ORIENTED GATEWAY OVERVIEW

- Service-oriented Gateway reference design based on S32G
- NXP service-oriented gateway extends EV management and optimization capabilities with vehicle-to-cloud processing.
- 12x Ethernet, 18x CAN FD, 5x LIN, 1x FlexRay interfaces
- PCI Express x1 slot for system expansion and USB 2.0 OTG connector
- M.2 slots for SSD storage and AI/ML acceleration modules
- JTAG debug and Aurora trace support
- Rugged enclosure with integrated thermal management
